# Calculator

A simple calculator webapp that supports basic arithmetic operations (addition, subtraction, multiplication, division) on integers.

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript

## How to Run

### Option 1: Open in browser (no server)

Open `index.html` in any browser.

### Option 2: With Docker

From this directory:

```bash
docker compose up --build
```

Then visit [http://localhost:8080](http://localhost:8080).
