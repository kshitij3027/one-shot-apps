# Binary to Decimal

A simple webapp that lets users enter up to 8 binary digits (0s and 1s) and displays the decimal equivalent.

## Tech Stack
- HTML
- CSS
- JavaScript (vanilla)

## How to Run

### With Docker
```bash
docker compose up --build
```
Open [http://localhost:8080](http://localhost:8080) in your browser.

### Without Docker
Open `index.html` directly in your browser.
